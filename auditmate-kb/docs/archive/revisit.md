# Revisit

⚠️ **Note:** Only users with permission to **View Archive / Revisit** can access the Revisit page.

![Revisit Page](./images/revisit-overview.png "width=800 nozoom")  
**Access:** Top Menu → **Archive**, Left Menu → **Revisit**

## Overview

|  |
|---|
| The **Revisit** page allows users to view a part and its details as they were before being archived. This helps verify historical information, review previous configurations, and reference the part’s state prior to archival. <br><br> The page displays: <br> - Part information (Part ID, Name, Status) <br> - Linked operations or steps <br> - Associated Machine - Uploaded documents <br> - CNC Programs - Others|

## How to Use Revisit


|  |
|---|
| You can review a part and see its details as they were before archival. <br>- Go to the **Archive** section and select the part. <br> - Click **Revisit** icon to open the complete detail preview. <br> - Review all displayed information for reference or historical context. <br> - Use this view to understand the part’s previous state, including operations, metadata, and related records. |


## Important Notes

|  |
|---|
| - Archiving is permanent unless restored by an authorized user. <br> - Ensure that the part is no longer active or tied to ongoing processes. <br> - Double-check dependencies or related modules before confirming. |
