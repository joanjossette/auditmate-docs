# Manage Users

⚠️**Note:** Only users with **Manage Users / Assign User Roles and Permissions** can perform the following task.  
For more information about user roles and permissions, see [Manage User Roles](/docs/getting-started/user-roles).




![Users](./images/users.png "width=800 nozoom")    
**Access:** Top Menu → **Admin**, Left Menu → **Users**  

## Add User

|  |  |
|---|---|
| You can create a new user account and assign appropriate roles and permissions to control access within the system.  <br><br>- Click **Create** to open the **Create User** form.  <br> - Fill in the required user details such as **Name**, **Primary Facility**, **Email**, and other relevant information. <br> - Assign the appropriate **User Role** to define access permissions. <br> - Click **Save** to add the new user to the system. <br><br> Once saved, the user will appear in the **Users** list and can log in based on the assigned role and permissions.| ![Create User Form](./images/create-user-form.png "width=400") |


## Edit or Update a User

|  |
|---|
| You can modify an existing user’s information, such as role, facility, or contact details. <br><br>-  Click on the corresponding **Edit icon** of the user you want to update. <br> - Update the necessary details (e.g., name, role, email, or facility). <br> - Click **Save** to apply the changes. <br><br> Once saved, the updated user information will reflect immediately in the **Users** list.|

## Delete a User



|  |
|---|
| You can remove a user from the system when they no longer require access. <br><br>- Click the corresponding **Delete icon** of the user you want to delete. <br> - Confirm the action when prompted. <br><br> Deleting a user is permanent and cannot be undone. Ensure that the user no longer needs system access before proceeding.|
