# Main Screen Overview

The main screen is your primary workspace. Understanding its layout helps you navigate the system efficiently.




| **Section** | **Description** |
|------------------|-----------------|
| **Top Menu** | Provides access to core functions, navigation options, and system settings|
| **Left Menu** | Displays context-specific options for the selected top-menu item|
| **Left Panel** | Displays the list of created entities for the selected entity <br>(all Parts, Operations for a Part, Machines for an Operation, or CNC Programs for a Machine)|
| **Center Menu** | Provides quick navigation across process entities <br>(Part, Operation, Machine, CNC Program)|
| **Left Panel** | Displays the detailed information of the process entity selected|
