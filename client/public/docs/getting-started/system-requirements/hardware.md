# Hardware Requirements

Ensure the target machine meets the following minimum and recommended specifications:

| Component       | Minimum Requirement   | Recommended Requirement              |
| --------------- | --------------------- | ------------------------------------ |
| Processor (CPU) | Dual-core 2.0 GHz     | Quad-core 2.5 GHz or higher          |
| Memory (RAM)    | 8 GB                  | 16 GB or higher                      |
| Storage         | 50 GB available space | 100 GB+ SSD                          |
| Disk Type       | HDD                   | SSD (for better performance)         |
| Network         | 1 Gbps LAN            | Stable high-speed network connection |
| Display         | 1366 × 768 resolution | 1920 × 1080 resolution               |

---

## Additional Notes

- Allocate additional storage for:
  - Database growth
  - File storage (uploads, logs, backups)
- SSD is highly recommended for:
  - Faster database performance
  - Improved application responsiveness
- Ensure reliable network connectivity for:
  - Remote access
  - API communication

> ⚠️ For production environments, always use the **recommended specifications** or higher to ensure optimal performance and scalability.

## Next Steps

After confirming that your hardware meets the requirements:

- Review the [Software Requirements](/docs/getting-started/system-requirements/software) to ensure all necessary software is installed
- Then proceed with the [Installation Guide](/docs/getting-started/installation) for step-by-step setup instructions
