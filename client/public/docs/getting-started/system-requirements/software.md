# Software Requirements

Before starting the installation, ensure the following software is installed and up to date:

| Category                 | Software / Version                       | Notes                                     |
| ------------------------ | ---------------------------------------- | ----------------------------------------- |
| Operating System         | Windows Server 2022 / 2025 or Windows 11 | Ensure latest updates are applied         |
| Database Server          | SQL Server 2022 or later                 | Mixed Mode Authentication required        |
| Database Tools           | SQL Server Management Studio (SSMS)      | Used for database configuration           |
| SQL Server Configuration | SQL Server Configuration Manager         | Used to enable TCP/IP and configure ports |
| Web Server               | Internet Information Services (IIS) 10+  | Installed via Server Manager              |
| .NET Framework / Runtime | .NET Framework 4.8 or higher             | Required for ASP.NET applications         |
| Web Browser              | Chrome, Edge, or Firefox                 | For testing the deployed website          |
| Firewall / Security Tool | Windows Defender Firewall                | Configure inbound rules for remote access |

> ⚠️ Make sure all required software is installed before proceeding with the installation steps.

## Next Steps

Once you have confirmed that your system meets the requirements, proceed with the installation:

- Follow the [Installation Guide](/docs/getting-started/installation) for step-by-step setup instructions
