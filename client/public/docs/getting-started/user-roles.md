# User Roles

Admins can add new roles to the system and manage the permissions assigned to each role. This allows them to control which users can access specific features, perform actions, or view information within the system, ensuring proper access control and security.

![Main Screen](./images/user-roles.png "width=800 nozoom")

- **Access:** Top Menu → **Admin**, Left Menu → **Permissions**  
- **Add Role:** Click **+ Add New Role** and enter the role name.  
- **Edit Permissions:** Check or uncheck boxes to grant or remove permissions.  
- **Save Changes:** Click the **Save** button to apply updates.
