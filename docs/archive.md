# Archive

The archive system preserves the complete version history of all machine programs and related documents, supports a structured backup strategy, and provides a revisit function for restoring or reviewing previous versions.

![Archive](./images/archive.png "width=800 nozoom")    
**Access:** Top Menu → **Archive**

## Archive Tools

Each archived part and operation row includes a set of action icons for managing the record. The Machine Documents and Machine Programs lists each have their own download and blacklist actions per item.

| Icon | Tool | Description |
|:---:|------|-------------|
| <img class="icon-inline" src="/images/archive-icon-list.svg" alt="List icon"/> | **List** | Default view; displays archived parts and operations in a searchable table. |
| <img class="icon-inline" src="/images/archive-icon-revisit.svg" alt="Revisit icon"/> | **Revisit** | Opens the full detail preview of a part as it existed before archiving, including linked operations, machines, documents, and machine programs. |
| <img class="icon-inline" src="/images/archive-icon-preview.svg" alt="View History icon"/> | **View History** | Displays the history logs of the archived part or operation. |
| <img class="icon-inline" src="/images/archive-icon-documents.svg" alt="Download Documents icon"/> | **Download Documents** | Downloads the documents attached to the archived part or operation. |
| <img class="icon-inline" src="/images/archive-icon-restore.svg" alt="Restore icon"/> | **Restore** | Bring an archived part or operation back to active status. |
| <img class="icon-inline" src="/images/archive-icon-blacklist.svg" alt="Blacklist icon"/> | **Blacklist** | Prevent an archived part, operation, machine document, or machine program from being restored or reused. |
| <img class="icon-inline" src="/images/archive-icon-download.svg" alt="Download icon"/> | **Download** | Downloads a copy of the individual machine document or machine program file. |
