# Main Screen Overview

The main screen is your primary workspace. Understanding its layout helps you navigate the system
![Main Screen](./images/main-screen-overview.png "width=800 nozoom")

| **Section** | **Description** |
|------------------|-----------------|
| **Top Menu** | Provides access to core functions, navigation options, and system settings|
| **Left Menu** | Displays context-specific options for the selected top-menu item|
| **Left Panel** | Displays the list of created entities for the selected entity <br>(all Parts, Operations for a Part, Machines for an Operation, or Machine Programs for a Machine)|
| **Center Menu** | Provides quick navigation across process entities <br>(Part, Operation, Machine, Machine Program)|
| **Center Panel** | Displays the detailed information of the process entity selected|
