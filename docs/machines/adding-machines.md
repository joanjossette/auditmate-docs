# Adding Machines

There are two ways to add a machine under a Machine Type:

### Option 1: Using the Create Button

<div class="doc-flex-row">
<div>

1\. On the Machines page, click the **Create** button and select **Machine** from the dropdown. <br> 2. The **Choose Machine Type** form will appear, displaying all machine types available in your facility/facilities. <br> 3. Select the machine type where the new machine will be created, then click **Next**. <br> 4. The **Create Machine** form will appear.

</div>
<div>

![Create Machine Form](./images/choose-machine-type-form.png "width=400")

</div>
</div>

### Option 2: Using the + Add Machine Button

<div class="doc-flex-row">
<div>

1\. Expand the machine type where the new machine will be created. <br> 2. Click the **+ Add Machine** button. <br> 3. The **Create Machine** form will appear.

</div>
<div>

![Add Machine Button](./images/add-machine.png "width=400")

</div>
</div>

---

## Create Machine Form

Depending on the machine head count of the selected Machine Type, the **Create Machine** form that will be shown will vary. <br><br>

<div class="doc-flex-row">
<div>

**Create Machine Form (Single-Head)**![Create Machine - Single Head](./images/create-machine-single-head.png "width=400")

</div>
<div>

**Create Machine Form (Multi-Head)**![Create Machine - Multi Head](./images/create-machine-multi-head.png "width=400")

</div>
</div>

1. Fill out the machine details <br> 
2. Click **Save** to create the machine. <br><br>

 ⚠️**Note:** Any new machine added to a Machine Type already linked to existing operation(s) will be automatically associated with those operations.
